# 教师备注

第 14 课引入了惯用 C++ 的一个重要原则：

```
Class
      ->
Objects
      ->
Interfaces
      ->
Natural Syntax
```

学生应该理解，**运算符重载不是为了让代码更短**。它的目的是让用户自定义类型的行为直观，并与内置类型保持一致。

关键收获是：

> **只有当运算符行为自然、可预测，并保留原运算符含义时，才应该重载运算符。**
> 

本课还介绍了 **友元函数**，它是一种受控地授予私有数据访问权限的机制。学生应该理解，友元关系是一种设计工具，而不是良好封装的替代品。这些概念为更高级主题做准备，例如拷贝语义、模板、泛型编程，以及设计可复用的 C++ 库。
