# 教师备注

第 7 课引入了一个重要转变：

> **知识有价值并不只是因为它被存储起来；当它能够被高效检索时，才真正变得有价值。**
> 

概念演进是：

```
变量
    ->
持久化
    ->
结构化笔记
    ->
搜索
```

学生应该认识到，搜索是 Google Search、Notion Search、Obsidian Search、IDE 代码搜索、数据库和 AI 检索系统等现代系统的基础。它是从简单存储走向智能知识管理的第一步。
