# 预处理命令（Preprocessor Commands）

预处理命令通常会改变最终参与编译的代码内容。

- `#include`
  - `#include` 告诉**预处理器（preprocessor）**将另一个文件的内容插入当前文件。下面插入的是 `iostream`，它提供输入/输出功能。

```cpp
#include <iostream>

int main()
{
    std::cout << "Hello, world!\n";
    return 0;
}
```

# 数组（Array）

```cpp
class Note;

Note ns[3];
vector<Note> nv(3);

ns[0].display();
nv[2].display();

// 数组下标越界
ns[3].display(); // 可能看似能运行，但属于未定义行为
```

```cpp
Note ns[3];  // 创建 3 个 Note 对象实例
Note* ns[3]; // 不创建 Note 对象，只创建 3 个指针
```
