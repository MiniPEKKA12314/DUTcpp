# C++ STL 算法入门

**C++；初学者路线；STL**

STL 算法是可复用函数，可对各种容器执行查找、排序、统计、变换等操作，无需反复手写相同循环。

# 什么是 STL 算法？

STL 算法通常定义在 `<algorithm>` 或 `<numeric>` 中，并对一段元素范围操作。范围通常由两个迭代器表示：

```cpp
algorithmName(container.begin(), container.end(), /* 额外参数 */);
```

- `begin()`：指向第一个元素。
- `end()`：指向最后一个元素的后一个位置。

这种 `[begin, end)` 的“左闭右开区间”模式贯穿 STL。

手写循环当然可以，但标准算法通常更短、更清晰，也能减少下标越界和边界条件错误。例如：

```cpp
std::sort(values.begin(), values.end());
```

一眼即可看出目标是排序。

# 四类常用算法

| 类别 | 作用 | 例子 |
|---|---|---|
| 查询 | 只检查，不修改数据 | `find`、`count`、`all_of` |
| 修改 | 原地改变元素 | `reverse`、`remove`、`transform` |
| 排序/重排 | 调整元素顺序 | `sort`、`unique` |
| 数值 | 根据范围计算值 | `accumulate`、`min_element` |

# `std::sort`：排序

**定义：**将范围按升序排列，也可传入自定义规则。

```cpp
#include <algorithm>
#include <iostream>
#include <vector>

int main()
{
    std::vector<int> nums = {5, 2, 9, 1, 5};
    std::sort(nums.begin(), nums.end());

    for (int number : nums) {
        std::cout << number << " ";
    }
}
```

输出：

```text
1 2 5 5 9
```

降序：

```cpp
#include <functional>

std::sort(nums.begin(), nums.end(), std::greater<int>());
```

# `std::find`：查找一个值

**定义：**扫描范围，返回第一个匹配元素的迭代器；找不到时返回 `end()`。

```cpp
#include <algorithm>
#include <iostream>
#include <string>
#include <vector>

int main()
{
    std::vector<std::string> fruits = {"apple", "pear", "kiwi"};
    auto it = std::find(fruits.begin(), fruits.end(), "pear");

    if (it != fruits.end()) {
        std::cout << "Found at index " << (it - fruits.begin());
    } else {
        std::cout << "Not found";
    }
}
```

输出：

```text
Found at index 1
```

注意：必须先比较 `it != fruits.end()`，才能使用 `*it`。对“未找到”的 `end()` 解引用是未定义行为。

`std::find` 与字符串成员函数 `string::find` 不同：

```cpp
note.find("C++");            // 在一个字符串中找子字符串
std::find(v.begin(), v.end(), 7); // 在容器中找一个元素
```

# `std::count`：统计匹配数量

**定义：**统计范围内等于指定值的元素数量。

```cpp
#include <algorithm>
#include <iostream>
#include <vector>

int main()
{
    std::vector<int> votes = {1, 2, 1, 1, 3, 2};
    int ones = std::count(votes.begin(), votes.end(), 1);
    std::cout << "Candidate 1 got " << ones << " votes";
}
```

输出：

```text
Candidate 1 got 3 votes
```

# `std::accumulate`：累计/求和

**定义：**把一段范围不断合并成一个结果；默认是累加。它位于 `<numeric>`，不是 `<algorithm>`。

```cpp
#include <iostream>
#include <numeric>
#include <vector>

int main()
{
    std::vector<int> prices = {10, 20, 15};
    int total = std::accumulate(prices.begin(), prices.end(), 0);
    std::cout << "Total: " << total;
}
```

输出：

```text
Total: 45
```

最后的 `0` 是初始值。也可传入自定义运算，例如从 `1` 开始并使用乘法得到乘积。

# `std::min_element` 与 `std::max_element`

**定义：**返回最小或最大元素的迭代器。

```cpp
#include <algorithm>
#include <iostream>
#include <vector>

int main()
{
    std::vector<int> temps = {72, 68, 90, 55};
    auto hottest = std::max_element(temps.begin(), temps.end());
    std::cout << "Hottest: " << *hottest;
}
```

输出：

```text
Hottest: 90
```

算法返回的是位置，所以要用 `*hottest` 解引用得到实际值。空容器时结果会等于 `end()`，使用前应先检查。

# `std::reverse`：反转顺序

**定义：**原地反转范围内的元素顺序。

```cpp
#include <algorithm>
#include <iostream>
#include <vector>

int main()
{
    std::vector<char> letters = {'c', 'a', 't'};
    std::reverse(letters.begin(), letters.end());

    for (char character : letters) {
        std::cout << character;
    }
}
```

输出：

```text
tac
```

# `std::transform`：逐个变换元素

**定义：**对范围中每个元素应用函数，并把结果写到目标位置。

```cpp
#include <algorithm>
#include <iostream>
#include <vector>

int main()
{
    std::vector<int> nums = {1, 2, 3, 4};

    std::transform(nums.begin(), nums.end(), nums.begin(),
                   [](int n) { return n * n; });

    for (int number : nums) {
        std::cout << number << " ";
    }
}
```

输出：

```text
1 4 9 16
```

其中：

```cpp
[](int n) { return n * n; }
```

是 lambda，即就地定义的小型匿名函数。

两个容器相加：

```cpp
#include <algorithm>
#include <iterator>
#include <vector>

std::vector<int> a = {1, 2, 3};
std::vector<int> b = {10, 20, 30};
std::vector<int> sum;

std::transform(a.begin(), a.end(), b.begin(), std::back_inserter(sum),
               [](int x, int y) { return x + y; });
// sum 为 {11, 22, 33}
```

### `transform` 与 `for_each`

若目标是根据旧值产生新值，使用 `transform` 语义更清楚：

```cpp
std::transform(values.begin(), values.end(), values.begin(),
               [](int value) { return value * 2; });
```

`for_each` 更适合执行动作，例如打印、记录日志或更新外部统计量：

```cpp
std::for_each(values.begin(), values.end(),
              [](int value) { std::cout << value << '\n'; });
```

# 其他常用算法

| 算法 | 简述 |
|---|---|
| `binary_search` | 在**已排序**范围中检查某值是否存在，复杂度 `O(log n)`。 |
| `unique` | 将连续重复元素移到后方，通常先排序；它本身不缩小容器。 |
| `for_each` | 对每个元素执行一个函数。 |
| `copy` | 将一个范围复制到目标位置。 |
| `remove` | 将不需要的元素移到末尾，但不改变容器大小。 |
| `all_of` / `any_of` | 检查是否全部/任意元素满足条件。 |

# 常见陷阱：`remove` 不会真正删除元素

```cpp
std::vector<int> nums = {1, 2, 3, 2, 5};
std::remove(nums.begin(), nums.end(), 2);
```

`remove` 只移动元素，返回新的逻辑结尾，容器的 `size()` 不会自动变小。真正删除要使用 erase-remove 惯用法：

```cpp
nums.erase(std::remove(nums.begin(), nums.end(), 2), nums.end());
// nums 现在是 {1, 3, 5}
```

# 练习建议

给定：

```cpp
std::vector<int> nums = {4, 1, 7, 3, 9, 2};
```

分别使用一个算法排序、一个算法求总和、一个算法检查 `7` 是否存在。尽量只用三行算法调用，不手写循环。
