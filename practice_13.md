# 教师备注

第 13 课引入了一个概念：**对象有生命周期**，不仅仅有数据和行为。

演进过程是：

```
Class
      ->
Object
      ->
Construction
      ->
Lifetime
      ->
Destruction
```

学生应该理解，构造函数和析构函数不是普通成员函数。它们定义了 **对象如何进入存在状态，以及如何离开存在状态**。

关键收获是：

> **设计良好的对象出生时就处于有效状态，并在生命周期结束时自动清理自己的资源。**
> 

本课为更高级的 C++ 主题做准备，例如初始化列表、拷贝构造函数、移动语义、动态内存管理、智能指针和资源安全的类设计。
