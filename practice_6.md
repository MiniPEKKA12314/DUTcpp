# 教师备注

本课表示以下转变：

| 前一版本 | 重点 |
| --- | --- |
| CLI Notes v1 | 运行期间的持久性 |
| CLI Notes v2 | 跨程序运行的持久性 |
| CLI Notes v3 | 有意义的结构化笔记内容 |

关键概念收获是：

> **一个记笔记系统变得有用，不只是因为它能记住信息，还因为它能以自然的形式捕获并组织人的想法。**
> 

本课为后续涉及笔记集合、搜索、标签、元数据和知识管理系统的版本做准备。
