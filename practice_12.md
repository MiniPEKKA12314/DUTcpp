将 CLI Notes 项目的一部分重构为 `Note` 类。

建议设计：

```

{
private:
string title;
string content;
string tag;

public:
void create();
void display()const;
void save()const;
void load();
};
```

要求：

- 将笔记数据存储为 private 成员。
- 使用成员函数操作对象。
- 消除全局笔记变量。
- 让 `main()` 只负责控制程序流程。

# 教师备注

第 12 课标志着课程中最重要的转变之一：

```
Variables
      ->
Functions
      ->
Data Flow
      ->
Classes
      ->
Objects
```

学生应该认识到，OOP **不只是学习新语法**。相反，它引入了一种关于软件设计的新思维方式：

- **过程式编程** 围绕 **函数** 组织软件。
- **面向对象编程** 围绕 **对象** 组织软件，对象结合了 **状态（数据）** 和 **行为（函数）**。

关键收获是：

> **类对一个概念建模，而对象表示这个概念的一个具体实例。**
> 

本课为后续主题打下基础，例如构造函数、访问控制、对象生命周期、组合、继承和多态。
