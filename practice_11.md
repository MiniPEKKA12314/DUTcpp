# 教师备注

第 11 课引入了第一个真正的内存层抽象：

```
Variable
    ->
Reference
    ->
Pointer
    ->
Memory Model
```

学生应该理解，指针不仅仅是一种语法特性。它们提供了一种理解数据如何存在于内存中，以及程序如何访问这些数据的思维模型。

关键收获是：

> **变量存储一个值，而指针存储这个值所在的位置。**
> 

本课为以下内容做准备：

- 动态内存
- 数组和指针算术
- 链式数据结构
- 智能指针
- 面向对象编程
- 系统级编程

这些内容都直接建立在指针和内存概念之上。
