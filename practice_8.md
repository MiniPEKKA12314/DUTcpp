## 问题 5 - CLI Notes v5（挑战）

在 CLI Notes v4 的基础上添加元数据支持。

菜单：

```
1. Create Note
2. View Note
3. Save Note
4. Load Note
5. Search Content
6. Search Tag
7. Exit
```

要求：

- 存储：
    - Title
    - Tag
    - Content
- 保存并加载所有信息。
- 使用 `find()` 搜索笔记内容。
- 直接搜索标签。
- 以结构化格式显示笔记。

示例：

```
===== NOTE =====

Title: Machine Learning Basics
Tag: AI

Content:
Supervised learning uses labeled data.

================
```

# 教师备注

第 8 课引入了一个重大的概念转变：

```
存储信息
       ->
查找信息
       ->
组织信息
```

学生应该理解，元数据不仅仅是额外信息，它是信息架构的基础。

关键收获是：

> **内容存储知识；元数据组织知识。**
> 

本课为后续主题做准备，例如笔记集合、过滤、笔记之间的关系、知识图谱和数据库设计。
