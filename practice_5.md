# 教师备注

本测评强调以下概念转变：

| 版本 | 能力 |
| --- | --- |
| CLI Notes v1 | 运行时记住内容 |
| CLI Notes v2 | 关闭后仍记住内容 |

学生应该理解，第 5 课并不仅仅是关于文件语法。更深层的思想是：

> **持久化让软件能够拥有超越单次执行的记忆。**
>
